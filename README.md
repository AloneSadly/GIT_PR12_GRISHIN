# MAX

## **MAX** - Российский мессенджер и цифровая платформа, запущенная 2025 году компанией VK. Позиционируется как *"Национальный мессенджер"* - универсальное мобильное приложение, которое по мимо обмена сообщениями и голосовой связи позволяет получать электронные государственные услуги

[аываы]: MAX.jpg

## Описание программы
```
1.Личные и групповые чаты
  - с поддержкой фото и видео
  - отправкой документов и файлов
```
```
2.Бесплатные голосовые и видео звонки
  - Работают даже на слабом интернете
  - Групповые беседы не ограничены
  - Ловит даже на парковке
```
```
3.Голосовые сообщения, стикеры и реакции
    - Для быстрого общения
```
```
4.Мини-приложения и боты
    - можно создать интернет-магазин, меню ресторана
    или сервис доставки прямо в мессенджере
```
```
5.GigaChat(ИИ-ассистент)
    - генерация текстов и картинок
    - пересказ статей 
    - расшифровка аудио
```
```
6.Интеграция с Госуслугами
    - опалата штрафов
    - подписание документов
    -обращение в госорганы
```

## Необходимые уловия

### **Android**: версия 10.0 и выше.
### **IOS**: версия 15 и выше
> [!WARNING]
> Для регистрации в MAX нужен российский или белорусский номер мобильного телефона.
> Регистрация по виртуальному или подменному номеру НЕВОЗМОЖНА.
